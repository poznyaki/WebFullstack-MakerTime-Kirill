# Звіт про виконану роботу: Музична платформа (Maker Time)

## 1. Виправлення Бекенду (Server)
- **Міграція на Express:** Сервер переписано з Fastify на Express відповідно до вимог проекту.
- **Multer:** Налаштовано завантаження MP3-файлів та обкладинок з валідацією типів та розміру.
- **Статичні файли:** Налаштовано роздачу завантажених файлів через `/static`.
- **База даних:** Реалізовано збереження метаданих треків (назва, автор, жанр, шляхи до файлів) у MySQL.
- **Авто-ініціалізація:** Створено скрипт `create-db.js` для автоматичного створення бази даних та таблиць.

## 2. Виправлення Бази Даних (Database)
- **Схема:** Виправлено `init.sql` (замінено PostgreSQL `BIGSERIAL` на MySQL `BIGINT AUTO_INCREMENT`).
- **Структура:** Створено таблиці `users` та `tracks` з відповідними індексами для пошуку.

## 3. Розробка Фронтенду (Client)
- **Технології:** React + Axios.
- **Функціонал (Level 1):**
    - Форма завантаження файлів з підтримкою `multipart/form-data`.
    - Список треків, що оновлюється в реальному часі після завантаження.
    - Аудіоплеєр, закріплений знизу (Player Bar), що відтворює потік з сервера.
- **Стилізація:** Додано базовий адаптивний CSS (`App.css`) з сіткою (Grid) для карток треків.

## 4. Виправлення оточення
- **Dependencies:** Додано відсутні пакети (`axios`, `cors`, `multer`, `concurrently`).
- **Environment:** Оновлено `.env` та виправлено помилки з нативними бінарними файлами Vite.
